使用指南

一、 可以是两种格式：1、外容器的子标签为li，<ul><li><h3>title1</h3><div>content1</div><h3>title2</h3><div>content2</div></li></ul>
                     2、除 1、 格式外，以外容器内两两为一组，一组中的第一个为头部title，第二个为内容区（如下test选项源代码）。
二、与jquery 的效果组合事，jquery 的效果放在后面，否则达不到预期效果。如: $(".accordion2").accordion().hide().show(2000);

参数列表：

        fit : true,            //为真时，内容区自动填充高度，以适应外容器
	autoHeight : false,    //fit:true时，此项恒为false;fit:false时,若外容器高度小于所有头部总高度时,此项为true,其它情况为入口值或默认值
	autoDevelop : false,    //为真时,可展开多项,fit:true时，此项为false,其它情况为入口值或默认值
	collapsible : false,    //为真时,展开项再次点击可关闭，反之不能关闭。
	animationTime : 500,    //完全展开与完全关闭所需的时间，因此，值越大，速度越慢，（毫秒）
	setSelected : 0,//初始化时展开的项，若为数字，则展开对应位置项；若为字符串，则由jquery选择器选取对应的头部项展开内容。（fit:false并且autoDevelop:false时才为多项展开）
	headers : {//可自定义头部样式...
		defaultClass : "msky-lap-header-default radius-top radius-bottom ",// 初始化时样式。
		hoverClass : "msky-lap-header-hover",//鼠标在上方时样式。
		activeClass : "msky-lap-header-active"//展开项的头部样式。
	},
	icons : {//可自定义图标样式...
		closedClass : "msky-icon-triangle-1-e",//选项关闭时的图标样式。
		openClass : "msky-icon-triangle-1-s"//
	},
	contents : {//可自定义内容框样式...
		defaultClass : "msky-lap-content-default radius-bottom",//activeClass 为空时的样式。
		activeClass : "msky-lap-content-active"// 内容展开的样式。
	}
